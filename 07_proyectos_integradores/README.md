# 🚀 07 · Proyectos Integradores

Esta sección contiene proyectos end-to-end que combinan múltiples habilidades: colección de datos, EDA, visualización y modelado.

## Qué contiene un proyecto integrador

Cada proyecto aquí sigue el ciclo completo de análisis:

1. **Definición del problema** — ¿Qué queremos responder?
2. **Obtención de datos** — Datasets, APIs, scraping
3. **Limpieza y preparación** — EDA, manejo de nulos
4. **Análisis y modelado** — Estadística, ML
5. **Visualización y comunicación** — Resultados claros
6. **Conclusiones** — Hallazgos e insights
